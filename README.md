# Square CI

### Introduction

Square CI is a tiny continuous integration tool based on [GitHub](https://github.com/).

### Others

- All code files are edited by [PyCharm](https://www.jetbrains.com/pycharm/).
- All ".md" files are edited by [Typora](http://typora.io).
